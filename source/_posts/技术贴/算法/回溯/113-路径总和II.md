## 题目描述

[113. 路径总和 II](https://leetcode-cn.com/problems/path-sum-ii/)

难度中等656

给你二叉树的根节点 `root` 和一个整数目标和 `targetSum` ，找出所有 **从根节点到叶子节点** 路径总和等于给定目标和的路径。

**叶子节点** 是指没有子节点的节点。

 <!-- more -->

```
输入：root = [5,4,8,11,null,13,4,7,2,null,null,5,1], targetSum = 22
输出：[[5,4,11,2],[5,8,4,5]]
```


 

**提示：**

- 树中节点总数在范围 `[0, 5000]` 内
- `-1000 <= Node.val <= 1000`
- `-1000 <= targetSum <= 1000`

## 解题思路及代码

```go
/**
 * Definition for a binary tree node.
 * type TreeNode struct {
 *     Val int
 *     Left *TreeNode
 *     Right *TreeNode
 * }
 */
// 回溯
func pathSum(root *TreeNode, targetSum int) [][]int {
    ans := make([][]int, 0)
    var help func(root *TreeNode, sum int, path *[]int)
    help = func(root *TreeNode, sum int, path *[]int) {
        // 叶子结点，递归终止
        if root == nil {
            return
        }
        *path = append(*path, root.Val)
        // 计算当前和与目标和的关系
        if root.Left == nil && root.Right == nil && sum + root.Val == targetSum {
            // 注意必须要复制一份(append会复制)，不然后面回溯会改变这个结果（slice是引用）
            ans = append(ans, append([]int(nil), *path...))         // 统计结果
        }
        help(root.Left, sum+root.Val, path)
        help(root.Right, sum+root.Val, path)
        // 回溯
        *path = (*path)[:len(*path)-1]
    }
    help(root, 0, &[]int{})
    return ans
}
```
