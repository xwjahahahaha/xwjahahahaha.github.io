## 题目描述

[904. 水果成篮](https://leetcode-cn.com/problems/fruit-into-baskets/)

难度中等125

在一排树中，第 `i` 棵树产生 `tree[i]` 型的水果。
你可以**从你选择的任何树开始**，然后重复执行以下步骤：

1. 把这棵树上的水果放进你的篮子里。如果你做不到，就停下来。
2. 移动到当前树右侧的下一棵树。如果右边没有树，就停下来。

请注意，在选择一颗树后，你没有任何选择：你必须执行步骤 1，然后执行步骤 2，然后返回步骤 1，然后执行步骤 2，依此类推，直至停止。

你有两个篮子，每个篮子可以携带任何数量的水果，但你希望每个篮子只携带一种类型的水果。

用这个程序你能收集的水果树的最大总量是多少？

<!-- more --> 

**示例 1：**

```
输入：[1,2,1]
输出：3
解释：我们可以收集 [1,2,1]。
```

**示例 2：**

```
输入：[0,1,2,2]
输出：3
解释：我们可以收集 [1,2,2]
如果我们从第一棵树开始，我们将只能收集到 [0, 1]。
```

**示例 3：**

```
输入：[1,2,3,2,2]
输出：4
解释：我们可以收集 [2,3,2,2]
如果我们从第一棵树开始，我们将只能收集到 [1, 2]。
```

**示例 4：**

```
输入：[3,3,3,1,2,1,1,2,3,3,4]
输出：5
解释：我们可以收集 [1,2,1,1,2]
如果我们从第一棵树或第八棵树开始，我们将只能收集到 4 棵水果树。
```

## 解题思路及代码

```go
// 滑动窗口
func totalFruit(fruits []int) int {
    l, r, kindMap, maxLen := 0, 0, make(map[int]int), 0
    for r < len(fruits) {
        // 1.如果当前数未出现，就将其加上
        kindMap[fruits[r]] += 1                 // map的值表示一种数字类别出现的个数
        // 2. 判断是否缩小
        for l < len(fruits) && kindMap[fruits[l]] > 0 && len(kindMap) > 2 {
            fmt.Println(kindMap)
            // 3. 缩小
            kindMap[fruits[l]] -= 1              // 减少一个个数
            if kindMap[fruits[l]] == 0 {
                delete(kindMap, fruits[l])
            }
            l++
        }
        // 4. 此时类型 <= 2 比较最大长度
        if nowLen := r-l+1; nowLen > maxLen {
            maxLen = nowLen
        }
        r++
    }
    return maxLen
}
```

```go
func totalFruit(fruits []int) int {
    max := 0
    l, r := 0, 0
    count := make(map[int]int)

    for r < len(fruits) {
        count[fruits[r]] ++

        // 限制条件：种类<2，同时删除一个种类的时候要删除之前所有的直到l之前没有该数
        for l<=r && len(count) > 2 && count[fruits[l]] > 0 {
            count[fruits[l]] --
            if count[fruits[l]] == 0 {
                delete(count, fruits[l])            // 如果缩小到0，就删除
            }
            l++
        }

        if r-l+1 > max {
            max = r-l+1
        }
        
        r++
    }

    return max
}
```
