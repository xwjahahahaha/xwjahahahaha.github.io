## 题目描述

[76. 最小覆盖子串](https://leetcode-cn.com/problems/minimum-window-substring/)

难度困难1570

给你一个字符串 `s` 、一个字符串 `t` 。返回 `s` 中涵盖 `t` 所有字符的最小子串。如果 `s` 中不存在涵盖 `t` 所有字符的子串，则返回空字符串 `""` 。

 <!-- more -->

**注意：**

- 对于 `t` 中重复字符，我们寻找的子字符串中该字符数量必须不少于 `t` 中该字符数量。
- 如果 `s` 中存在这样的子串，我们保证它是唯一的答案。

 

**示例 1：**

```
输入：s = "ADOBECODEBANC", t = "ABC"
输出："BANC"
```

**示例 2：**

```
输入：s = "a", t = "a"
输出："a"
```

**示例 3:**

```
输入: s = "a", t = "aa"
输出: ""
解释: t 中两个字符 'a' 均应包含在 s 的子串中，
因此没有符合条件的子字符串，返回空字符串。
```

 

**提示：**

- `1 <= s.length, t.length <= 105`
- `s` 和 `t` 由英文字母组成

 

**进阶：**你能设计一个在 `o(n)` 时间内解决此问题的算法吗？



## 解题思路及代码

```go
// 滑动窗口
// 注意：这一题是当窗口中间状态一旦不满足（可能是缩小窗口导致）就立即滑动右边直到让其满足
func minWindow(s string, t string) string {
    ansL, minLen := -1, math.MaxInt32
    // 1. 将t转换为一个tMap, sMap记录当前覆盖程度
    sMap, tMap := make(map[byte]int, 0), make(map[byte]int, 0)
    for _, c := range t {
        tMap[byte(c)]+=1
    }
    // 2. 滑动窗口[l, r]，一旦满足就
    for l, r := 0, 0; r < len(s); r++ { // 创建右端不断右移
        // 判断当前字符是否覆盖
        if _, has := tMap[s[r]]; has {
            sMap[s[r]]+=1       // 存在则记录
        } 
        // 判断是否要缩减窗口
        for check(sMap, tMap) && l <= r {
            // 比较最小覆盖长度
            if (r-l+1) < minLen {
                minLen = r-l+1
                ansL = l
            }
            // 开始缩减
            if _, has := tMap[s[l]]; has {
                // 如果有就删除记录
                sMap[s[l]]-=1
            }
            l++
        }
    }
    if ansL == -1 {
        return ""
    }
    return s[ansL:ansL+minLen]
}

// 检查当前s子串是否覆盖t
func check(sMap, tMap map[byte]int) bool {
    for k, v := range tMap {
        if sMap[k] < v {
            return false
        }
    }
    return true
}
```
