## 题目描述

两个整数之间的汉明距离指的是这两个数字对应二进制位不同的位置的数目。

给出两个整数 x 和 y，计算它们之间的汉明距离。

注意：
0 ≤ x, y < 231.

示例:

输入: x = 1, y = 4

输出: 2

解释:
1   (0 0 0 1)
4   (0 1 0 0)
       ↑   ↑

上面的箭头指出了对应二进制位不同的位置。

## 解题思路和代码

通过将两个数XOR异或，得到的结果统计1的个数即可，因为只有两位不同异或才为1
那么原始问题就变成了位计数问题

```python
# 第一种方法：扫描判断对应位置的两个数是否相同，累计 
# 时间复杂度O(1)
class Solution(object):
    def hammingDistance(self, x, y):
        直接的一种：先求异或再计数
        z = bin(x ^ y)
        count = 0
        for index in range(2, len(z)):
            if z[index] == '1':
                count += 1
        return count
        这种直接调用语言自带函数的方法，在项目开发中可取，但是在leetcode上不可取
        return bin(x ^ y)[2:].count("1")
```

```python           
# 第二种方法: 移位计数 
# 时间复杂度O(1)
class Solution(object):
    def hammingDistance(self, x, y):
        # 右移位
        z = x ^ y
        count = 0
        while z > 0:
            # 先判断最后一位是否为1
            if z & 1 == 1 :
                count += 1
            # 移位
            z >>= 1
        return count
```

```python
# 第三种方法: 布赖恩·克尼根算法
# 算法利用 x & x-1 去除掉最后一位的1来快速计数
# 时间复杂度O(1)
class Solution(object):
    def hammingDistance(self, x, y):
        count = 0
        z = x ^ y
        while z:
            z &= (z-1)
            count += 1
        return count
```
