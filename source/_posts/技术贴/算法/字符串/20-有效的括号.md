## 20. 有效的括号

20. 有效的括号

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。

左括号必须以正确的顺序闭合。

<!-- more -->

注意空字符串可被认为是有效字符串。

示例 1:

输入: "()"

输出: true

示例 2:

输入: "()[]{}"

输出: true

示例 3:

输入: "(]"

输出: false

示例 4:

输入: "([)]"

输出: false

示例 5:

输入: "{[]}"

输出: true


## 解题思路及代码

1. 栈结构抵消

时间复杂度O(N),空间复杂度O（N）

```java
class Solution {
    public boolean isValid(String s) {
        //初始情况筛除
        int len = s.length();
        if(s == null || len == 0) return true;
        if(len % 2 != 0) return false;
        //创建map
        Map<Character, Character> map = new HashMap<>();
        map.put('(', ')');
        map.put('[', ']');
        map.put('{', '}');
        //创建LinkedList
        LinkedList<Character> list = new LinkedList<>();
        //读串
        for (char c : s.toCharArray()) {
            //如果是左半边就添加到LinkedList
            if(c == '(' || c == '[' || c == '{'){
                list.add(c);
            }
            //不是的话就从栈中弹出元素比较
            //①如果读到右半边list是0（没有左半边）那么一定是false
            //②如果弹出的元素不匹配也是false
            else if(list.size() == 0 || map.get(list.removeLast()) != c) {
                return false;
            }
        }
        return list.size() == 0;
    }
}
```

2. 字符串替换
```python
class Solution(object):
    def isValid(self, s):
        """
        :type s: str
        :rtype: bool
        """
        while '()' in s or '[]' in s or '{}' in s:
            s = s.replace('()', '')
            s = s.replace('[]', '')
            s = s.replace('{}', '')
        if s == '':
            return True
        return False
```
