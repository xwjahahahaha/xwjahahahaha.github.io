## 题目描述

#### [125. 验证回文串](https://leetcode-cn.com/problems/valid-palindrome/)

难度简单

给定一个字符串，验证它是否是回文串，只考虑字母和数字字符，可以忽略字母的大小写。

**说明：**本题中，我们将空字符串定义为有效的回文串。

**示例 1:**

```
输入: "A man, a plan, a canal: Panama"
输出: true
```

**示例 2:**

```
输入: "race a car"
输出: false
```

## 解题思路及代码

```java
class Solution {
    //简单的思路:左右对撞双指针
    //时间复杂度O(s.length)
    //空间复杂度O(1)
    public boolean isPalindrome(String s) {
        int lo = 0, hi = s.length() - 1;
        while(lo < hi){
            //排除掉其他字符
            while(!Character.isLetterOrDigit(s.charAt(lo)) && lo < hi){
                ++ lo;
            }
            while(!Character.isLetterOrDigit(s.charAt(hi)) && lo < hi){
                -- hi;
            }
            if(Character.toLowerCase(s.charAt(lo)) != Character.toLowerCase(s.charAt(hi))){
                return false;
            }
            ++ lo;
            -- hi;
        }
        return true;
    }
}


class Solution {
    //思路2： 直接翻转比对
    public boolean isPalindrome(String s) {
        StringBuffer pureString = new StringBuffer();
        //提取有效字符组成纯字母数字字符数组
        for(int i=0; i<s.length(); i++){
            if(Character.isLetterOrDigit(s.charAt(i))){
                pureString.append(Character.toLowerCase(s.charAt(i)));
            }
        }
        //回文判断
        return pureString.toString().equals(pureString.reverse().toString());
    }
}
```
