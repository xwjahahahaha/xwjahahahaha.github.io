## 题目描述

#### [49. 字母异位词分组](https://leetcode-cn.com/problems/group-anagrams/)

给定一个字符串数组，将字母异位词组合在一起。字母异位词指字母相同，但排列不同的字符串。

**示例:**

```
输入: ["eat", "tea", "tan", "ate", "nat", "bat"]
输出:
[
  ["ate","eat","tea"],
  ["nat","tan"],
  ["bat"]
]
```

## 解题思路及代码

视频：https://leetcode-cn.com/problems/group-anagrams/solution/zi-mu-yi-wei-ci-fen-zu-by-leetcode-solut-gyoc/

==**在异构中寻找相同的点：**==

**1. 异构单词的字母出现次数相同,**

**2. 如果排序字母后,每个异构单词都是相同的**

**==本题是根据特征进行分类问题,那么就应该想到使用散列表(哈希表)==**

根据第一个特点可以构建唯一字母出现次数字符串,根据第二个特点可以先对单词的字母排序作为key

```golang
// 方法一：根据排序后的字母顺序构建散列表
func groupAnagrams(strs []string) [][]string {
    resMap := make(map[string][]string)
    res := [][]string{}
    // 1. 遍历数组
    for _, v := range strs {
        // 2. 对每个单词的字母排序 => key (使用sort.Slice)
        s := []byte(v)
        sort.Slice(s, func(i, j int) bool { return s[i] < s[j] })
        key := string(s)
        // 3. 相同的集合到value
        resMap[key] = append(resMap[key], v)
    }
    // 4. 格式转换输出
    for _, v := range resMap{
        res = append(res, v)
    }
    return res
}

// 方法二：构造数量字符串作为key构建散列表
func groupAnagrams(strs []string) [][]string {
    resMap := map[[26]int][]string{}
    // 1. 统计单词字母出现数量
    for _, v := range strs {
        // 2. 构建数量表达字符串 => key
        key := makeSortedSymbol(v)
        // 3. 累加相同key的字符串
        resMap[key] = append(resMap[key], v)
    }
    res := make([][]string, 0, len(resMap))
    // 4. 格式化结果
    for _, v := range resMap {
        res = append(res, v)
    }
    return res
}

func makeSortedSymbol(s string) [26]int{
    cnt := [26]int{} 
    for _, c := range s {
        cnt[c - 'a'] ++
    }
    return cnt
}
```
