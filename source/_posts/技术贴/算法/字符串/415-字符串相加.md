## 题目描述

[415. 字符串相加](https://leetcode-cn.com/problems/add-strings/)

难度简单

给定两个字符串形式的非负整数 `num1` 和`num2` ，计算它们的和并同样以字符串形式返回。

你不能使用任何內建的用于处理大整数的库（比如 `BigInteger`）， 也不能直接将输入的字符串转换为整数形式。

 <!-- more -->

**示例 1：**

```
输入：num1 = "11", num2 = "123"
输出："134"
```

**示例 2：**

```
输入：num1 = "456", num2 = "77"
输出："533"
```

**示例 3：**

```
输入：num1 = "0", num2 = "0"
输出："0"
```

## 解题思路及代码

```go
func addStrings(num1 string, num2 string) string {
    ret, carry := "", 0
    // 创建两个指针分别指向尾部
    p1, p2 := len(num1)-1, len(num2)-1  
    for p1 >= 0 || p2 >= 0 || carry > 0 {      
        // 注意carry > 0，即使两个指针都达到了最前面，可能还有进位没处理
        // 取值相加
        n1, n2 := 0, 0
        if p1 >= 0 {
            n1 = int(num1[p1] - '0')
        }
        if p2 >= 0 {
            n2 = int(num2[p2] - '0')
        }
        // 计算和
        sum := n1 + n2 + carry
        // 取进位
        carry = sum/10
        // 加到结果集前面
        ret = strconv.Itoa(sum%10) + ret
        // 同步向前移动
        p1--; p2--
    }
    return ret
}
```
