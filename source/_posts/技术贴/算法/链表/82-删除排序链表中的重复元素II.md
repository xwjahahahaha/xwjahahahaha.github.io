## 题目描述

[82. 删除排序链表中的重复元素 II](https://leetcode-cn.com/problems/remove-duplicates-from-sorted-list-ii/)

难度中等777

存在一个按升序排列的链表，给你这个链表的头节点 `head` ，请你删除链表中所有存在数字重复情况的节点，只保留原始链表中 **没有重复出现** 的数字。

返回同样按升序排列的结果链表。

<!-- more --> 

**示例 1：**

![img](http://xwjpics.gumptlu.work/qinniu_uPic/linkedlist1.jpg)

```
输入：head = [1,2,3,3,4,4,5]
输出：[1,2,5]
```

**示例 2：**

![img](http://xwjpics.gumptlu.work/qinniu_uPic/linkedlist2.jpg)

```
输入：head = [1,1,1,2,3]
输出：[2,3]
```

 

**提示：**

- 链表中节点数目在范围 `[0, 300]` 内
- `-100 <= Node.val <= 100`
- 题目数据保证链表已经按升序排列

## 解题思路及代码

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
// 双指针
// 需要加虚拟头节点，因为原本的head可能被删除
func deleteDuplicates(head *ListNode) *ListNode {
    if head == nil || head.Next == nil {
        return head
    }
    preHead := &ListNode{-101, head}
    low, fast := preHead, head
    for fast != nil {
        // fast先走,判断其下一个节点与自己是否重复,遇到重复就一直走
        for fast != nil && fast.Next != nil && fast.Val == fast.Next.Val {
            fast = fast.Next
        }
        // 判断是否需要删除
        if low.Next != fast {
            // 出现了重复,要删除,此时fast在重复的最后一个节点上 
            fast = fast.Next
            low.Next = fast
        }else {
            fast = fast.Next
            low = low.Next
        }
    }
    return preHead.Next
}
```
