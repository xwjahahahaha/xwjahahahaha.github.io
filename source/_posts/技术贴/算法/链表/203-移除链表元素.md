## 题目描述

[203. 移除链表元素](https://leetcode-cn.com/problems/remove-linked-list-elements/)

难度简单

给你一个链表的头节点 `head` 和一个整数 `val` ，请你删除链表中所有满足 `Node.val == val` 的节点，并返回 **新的头节点** 。

 <!-- more -->

**示例 1：**

![img](http://xwjpics.gumptlu.work/qinniu_uPic/removelinked-list.jpg)

```
输入：head = [1,2,6,3,4,5,6], val = 6
输出：[1,2,3,4,5]
```

**示例 2：**

```
输入：head = [], val = 1
输出：[]
```

**示例 3：**

```
输入：head = [7,7,7,7], val = 7
输出：[]
```

**提示：**

- 列表中的节点数目在范围 `[0, 104]` 内
- `1 <= Node.val <= 50`
- `0 <= val <= 50`

## 解题思路及代码

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func removeElements(head *ListNode, val int) *ListNode {
    if head == nil {
        return nil
    }
    // 使用一个虚拟头节点
    p := &ListNode{-1, head}
    head = p
    for p != nil {
        if p.Next != nil && p.Next.Val == val {
            p.Next = p.Next.Next
        }else{
            p = p.Next
        }
    }
    return head.Next            // 注意：这里是返回头节点的下一个，如果全部都是val则返回的就是nil
}
```

二刷：

```go
func removeElements(head *ListNode, val int) *ListNode {
    preHead := &ListNode{-1, head}
    p, t := preHead, preHead
    for p != nil {
        for t=p.Next; t!=nil && t.Val == val; t=t.Next {}
        p.Next = t 
        p = p.Next
    }

    return preHead.Next
}
```
