## 题目描述

[328. 奇偶链表](https://leetcode-cn.com/problems/odd-even-linked-list/)

难度中等

给定一个单链表，把所有的奇数节点和偶数节点分别排在一起。请注意，这里的奇数节点和偶数节点指的是节点编号的奇偶性，而不是节点的值的奇偶性。

请尝试使用原地算法完成。你的算法的空间复杂度应为 O(1)，时间复杂度应为 O(nodes)，nodes 为节点总数。

<!-- more -->

**示例 1:**

```
输入: 1->2->3->4->5->NULL
输出: 1->3->5->2->4->NULL
```

**示例 2:**

```
输入: 2->1->3->5->6->4->7->NULL 
输出: 2->3->6->7->1->5->4->NULL
```

**说明:**

- 应当保持奇数节点和偶数节点的相对顺序。
- 链表的第一个节点视为奇数节点，第二个节点视为偶数节点，以此类推。

## 解题思路及代码

**分离合并**的思想(但是不使用额外的空间)

时间复杂度O(N)

空间复杂度O(1)

![osTp4e](http://xwjpics.gumptlu.work/qinniu_uPic/osTp4e.jpg)

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
 // 分离合并
 // 将偶链表分离出来,在添加到奇链表之后
func oddEvenList(head *ListNode) *ListNode {
    if head == nil {
        return head
    }
    // 将链表分离为奇偶两个链表
    // 创建奇偶指针
    odd, even := head, head.Next
    // 记录偶链表的头
    evenHead := even
    // 一直读取,直到奇指针先结束
    // even.Next != nil 为了防止nil.Next的空指针出现
    for even != nil && even.Next != nil {
        // 先奇节点开始
        odd.Next = even.Next
        odd = odd.Next
        // 偶节点
        even.Next = odd.Next
        even = even.Next
    }
    // 拼接
    odd.Next = evenHead
    return head
}
```

二刷：

```go
func oddEvenList(head *ListNode) *ListNode {
    oddHead := &ListNode{}
    evenHead := &ListNode{}
    p1, p2 := oddHead, evenHead

    var p *ListNode
    i := 1
    for p=head; p!=nil; p=p.Next {
        if i % 2 != 0 {
            // 奇数
            p1.Next = p
            p1 = p1.Next
        }else {
            // 偶数
            p2.Next = p
            p2 = p2.Next
        }
        i++
    } 

    // 断开
    p1.Next, p2.Next = nil, nil

    // 链接
    for p1=oddHead; p1.Next!=nil; p1=p1.Next {}
    p1.Next = evenHead.Next

    return oddHead.Next 
}
```
