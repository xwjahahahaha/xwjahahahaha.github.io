## 题目描述

#### [2. 两数相加](https://leetcode-cn.com/problems/add-two-numbers/)

难度中等

给你两个 **非空** 的链表，表示两个非负的整数。它们每位数字都是按照 **逆序** 的方式存储的，并且每个节点只能存储 **一位** 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

<!-- more --> 

**示例 1：**

![img](https://assets.leetcode-cn.com/aliyun-lc-upload/uploads/2021/01/02/addtwonumber1.jpg)

```
输入：l1 = [2,4,3], l2 = [5,6,4]
输出：[7,0,8]
解释：342 + 465 = 807.
```

**示例 2：**

```
输入：l1 = [0], l2 = [0]
输出：[0]
```

**示例 3：**

```
输入：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
输出：[8,9,9,9,0,0,0,1]
```

## 解题思路及代码

时间复杂度O(max(m, n))

空间复杂度O(max(m,n))

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */

// 基础运算的规律: 两个数按位相加
// 当前位的计算: (n1 + n2 + carry) % 10
// 进位的计算: (n1 + n2 + carry) / 10
func addTwoNumbers(l1 *ListNode, l2 *ListNode) *ListNode {
   carry := 0
   var tail, head  *ListNode
   for l1 != nil || l2 != nil {
        // 取值, 如果已经取完那么默认就是0
        n1, n2 := 0, 0
        if l1 != nil {
           n1 = l1.Val
           l1 = l1.Next
        }
        if l2 != nil {
           n2 = l2.Val
           l2 = l2.Next
        }
        digit := (n1 + n2 + carry) % 10
        carry = (n1 +  n2 + carry) / 10
        if head == nil {
            // 第一处创建
            tail = &ListNode{Val : digit}
            head = tail
        }else {
            // 随后
            tail.Next = &ListNode{Val : digit}
            tail = tail.Next
        }
   }
   // 检查是否需要再进位. 这一步是关键
   if carry > 0 {
        tail.Next = &ListNode{Val : carry}
   }
   return head
}
```

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func addTwoNumbers(l1 *ListNode, l2 *ListNode) *ListNode {
    p1, p2 := l1, l2
    // 出栈计算
    head, carry := &ListNode{}, 0           // head是结果链表头
    p := head
    for p1 != nil  || p2 != nil {
        num1, num2 := 0, 0
        if p1 != nil {
            num1 = p1.Val
            p1 = p1.Next
        }
        if p2 != nil {
            num2 = p2.Val
            p2 = p2.Next
        }
        sum := num1 + num2 + carry
        // 计算进位
        carry = sum / 10
        newNode := &ListNode{sum % 10, nil}
        p.Next = newNode
        p = newNode
    }
    // 查看是否还有进位
    if carry > 0 {
        newNode := &ListNode{carry, nil}
        p.Next = newNode
    }
    return head.Next
}
```
