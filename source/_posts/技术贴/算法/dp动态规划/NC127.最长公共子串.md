## 题目描述

### 描述

给定两个字符串str1和str2,输出两个字符串的最长公共子串

题目保证str1和str2的最长公共子串存在且唯一。 

数据范围： $1 \le |str1|,|str2| \le 5000$
要求： 空间复杂度$ O(n^2)$，时间复杂度 $O(n^2)$

<!-- more -->

### 示例1

输入：

```
"1AB2345CD","12345EF"
```

返回值：

```
"2345"
```

### 备注：

```
1 \leq |str_1|, |str_2| \leq 5\,0001≤∣str1∣,∣str2∣≤5000
```

## 解题思路及代码

```go
package main

// import "fmt"
/**
 * longest common substring
 * @param str1 string字符串 the string
 * @param str2 string字符串 the string
 * @return string字符串
*/
// dp[i][j] 表示str1[:i](不取i), str2[:j]的最大公共子串
// 注意子串与子序列的区别，子串必须连续
// 最长公共子串是所有dp数组中最大值
func LCS( str1 string ,  str2 string ) string {
    m, n := len(str1), len(str2)
    // 初始化
    dp := make([][]int, m+1)
    for i:=0; i<=m; i++ {
        dp[i] = make([]int, n+1)
    }
    
    // 填表
    max, end := 0, 0      
    for i:=0; i<m; i++ {
        for j:=0; j<n; j++ {
            if str1[i] == str2[j] {
                dp[i+1][j+1] = dp[i][j]+1
                // 判断所有dp中的最大值
                if dp[i+1][j+1] > max {
                    end = i            // 记录最长子串的最后一个字符串的位置
                    max = dp[i+1][j+1]
                }
            }
            
        }
    }
    return str1[end-max+1:end+1]
}
```
