## 题目描述

<!-- more -->

[494. 目标和](https://leetcode.cn/problems/target-sum/)

难度中等1324

给你一个整数数组 `nums` 和一个整数 `target` 。

向数组中的每个整数前添加 `'+'` 或 `'-'` ，然后串联起所有整数，可以构造一个 **表达式** ：

- 例如，`nums = [2, 1]` ，可以在 `2` 之前添加 `'+'` ，在 `1` 之前添加 `'-'` ，然后串联起来得到表达式 `"+2-1"` 。

返回可以通过上述方法构造的、运算结果等于 `target` 的不同 **表达式** 的数目。

 

**示例 1：**

```
输入：nums = [1,1,1,1,1], target = 3
输出：5
解释：一共有 5 种方法让最终目标和为 3 。
-1 + 1 + 1 + 1 + 1 = 3
+1 - 1 + 1 + 1 + 1 = 3
+1 + 1 - 1 + 1 + 1 = 3
+1 + 1 + 1 - 1 + 1 = 3
+1 + 1 + 1 + 1 - 1 = 3
```

**示例 2：**

```
输入：nums = [1], target = 1
输出：1
```

 

**提示：**

- `1 <= nums.length <= 20`
- `0 <= nums[i] <= 1000`
- `0 <= sum(nums[i]) <= 1000`
- `-1000 <= target <= 1000`

通过次数256,009

提交次数521,107

## 解题思路及代码

```go
// 动态规划：01背包问题
// L - R = target
// L + R = sum
// L = (sum+target)/2
// 问题就变为：在nums中找寻和为L的一些数的种类个数 => 就是通过+/-运算符运算之后结果为target的种类数
func findTargetSumWays(nums []int, target int) int {
    n := len(nums)
    if n == 0 {
        return 0
    }

    sum := 0
    for _, num := range nums {
        sum += num
    }

    l := (sum+target)/2
    // 如果发生了下取整，那么一定就无法找到目标和为l的一系列数，因为和需要是一个小数，所以排除掉无意义
    if l*2 != sum+target || l < 0 {
        return 0
    }

    // dp[i][j]表示从[0,i)中取一些数的和为j的最大种类个数
    dp := make([][]int, n+1)
    for i:=0; i<=n; i++ {
        dp[i] = make([]int, l+1)
    }
     
    // 初始化
    dp[0][0] = 1
    
    for i:=1; i<=n; i++ {
        for j:=0; j<=l; j++ {
            if j >= nums[i-1] {
                // 能拿，不拿的种类数 + 拿的种类数
                dp[i][j] = dp[i-1][j] + dp[i-1][j-nums[i-1]]
            }else {
                // 不能拿
                dp[i][j] = dp[i-1][j]
            }
        }
    }
    
    return dp[n][l]
}
```
