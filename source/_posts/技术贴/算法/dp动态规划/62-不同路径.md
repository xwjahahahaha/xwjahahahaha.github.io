## 题目描述

[62. 不同路径](https://leetcode-cn.com/problems/unique-paths/)

难度中等984

一个机器人位于一个 `m x n` 网格的左上角 （起始点在下图中标记为 “Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为 “Finish” ）。

问总共有多少条不同的路径？

 <!-- more -->

**示例 1：**

![img](https://assets.leetcode.com/uploads/2018/10/22/robot_maze.png)

```
输入：m = 3, n = 7
输出：28
```

**示例 2：**

```
输入：m = 3, n = 2
输出：3
解释：
从左上角开始，总共有 3 条路径可以到达右下角。
1. 向右 -> 向下 -> 向下
2. 向下 -> 向下 -> 向右
3. 向下 -> 向右 -> 向下
```

**示例 3：**

```
输入：m = 7, n = 3
输出：28
```

**示例 4：**

```
输入：m = 3, n = 3
输出：6
```

## 解题思路及代码

```go
// dfs搜索
// m、n过大时超时
func uniquePaths(m int, n int) int {
    var count int
    var dfs func(int, int)
    dfs = func(i, j int){
        if i == m-1 && j == n-1 {
            count ++
            return 
        }
        // 向右走
        if j < n-1 {
            dfs(i, j+1)
        }
        // 向下走
        if i < m-1 {
            dfs(i+1, j)
        } 
    }
    dfs(0, 0)
    return count
}

// 动态规划
// 设走到当前位置的路径数是f(i, j)
// 那么其上一步一定是从f(i-1, j) 或者 f(i, j-1)走过来的
// => 递推式: f(i, j) = f(i-1, j) + f(i, j-1)
// 边界条件: dp[0][0] = dp[0][j] = dp[i][0] = 1
// 时间复杂度O(mn)
func uniquePaths(m int, n int) int {
    // 创建二维数组
    dp := make([][]int, m)
    for i := range dp {
        dp[i] = make([]int, n)
    }
    dp[0][0] = 1
    for i:=0; i<m; i++ {
        for j:=0; j<n; j++ {
            if i-1 >= 0 && j-1 >= 0 {       // 确保i-1、j-1有效
                dp[i][j] = dp[i-1][j] + dp[i][j-1]
            }else if i == 0 && j-1 >= 0{    // 边界条件: 第一行都为1
                dp[i][j] = dp[i][j-1]
            }else if j == 0 && i-1 >= 0{    // 边界条件: 第一列都为1
                dp[i][j] = dp[i-1][j]
            }
        }
    }
    return dp[m-1][n-1]
}
```

二刷：

```go
func uniquePaths(m int, n int) int {
    // 1. 定义dp
    // dp[i][j]表示到达(i, j)位置的路径个数
    dp := make([][]int, m)
    for i:=0; i<m; i++ {
        dp[i] = make([]int, n)
    }

    // 2. 递推公式
    // 只能从左、上面两个方向来
    // dp[i][j] = dp[i-1][j] + dp[i][j-1]

    // 3. 初始化
    // 初始化定义第一行和第一列的值都一定是1
    for i:=1; i<m; i++ {
        dp[i][0] = 1
    }
    for i:=1; i<n; i++ {
        dp[0][i] = 1
    }    
    dp[0][0] = 1            // 虽然不符合dp定义，但是符合题目样例

    // 4. 填表顺序, 查看递推公式，都是先左/上，所以必须先从左到右、从上到下，符合一般二维数组的遍历顺序
    for i:=1; i<m; i++ {
        for j:=1; j<n; j++ {
            // 中间
            dp[i][j] = dp[i-1][j] + dp[i][j-1]
        }
    }

    // 5. 计算结果
    return dp[m-1][n-1]
}
```
