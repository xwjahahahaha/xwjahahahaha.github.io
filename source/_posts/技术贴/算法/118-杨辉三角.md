## 题目描述

给定一个非负整数 numRows，生成杨辉三角的前 numRows 行。

在杨辉三角中，每个数是它左上方和右上方的数的和。

示例:

![](http://xwjpics.gumptlu.work/qiniu_picGo/20200705083346.png)

输入: 5

输出:
[
     [1],
    [1,1],
   [1,2,1],
  [1,3,3,1],
 [1,4,6,4,1]
]

## 解题思路及代码

``` python
# 自己的笨办法:迭代
# 时间复杂度O(N*N) 空间复杂度O(N*N)
class Solution(object):
    def generate(self, numRows):
        """
        :type numRows: int
        :rtype: List[List[int]]
        """
        if numRows == 0:
            return []
        if numRows == 1:
            return [[1]]
        resList = []
        list1 = [1]
        list2 = [1, 1]
        resList.append(list1)
        resList.append(list2)
        # 迭代添加
        for i in range(3, numRows+1):
            list1 = [1, 1]
            for j in range(1, i-1):
                list1.insert(j, list2[j]+list2[j-1])
            resList.append(list1)
            list2 = list1
        return resList
```
