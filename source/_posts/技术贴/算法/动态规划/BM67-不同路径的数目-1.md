## 问题描述

一个机器人在m×n大小的地图的左上角（起点）。

机器人每次可以向下或向右移动。机器人要到达地图的右下角（终点）。

可以有多少种不同的路径从起点走到终点？

![img](http://xwjpics.gumptlu.work/qinniu_uPic/873CB1F2327F70DA0CA0FDC797F894A7.png)

<!-- more -->

备注：m和n小于等于100,并保证计算结果在int范围内

数据范围：$0 < n,m \le 100$，保证计算结果在32位整型范围内

要求：空间复杂度 $O(nm)$，时间复杂度 $O(nm)$

进阶：空间复杂度 $O(1)$，时间复杂度 $O(min(n,m))$

### 示例1

输入：

```
2,1
```

返回值：

```
1
```

### 示例2

输入：

```
2,2
```

返回值：

```
2
```

## 解题思路及代码

```go
package main

/**
  * 
  * @param m int整型 
  * @param n int整型 
  * @return int整型
*/
// dp
func uniquePaths( m int ,  n int ) int {
    dp := make([][]int, m+1)
    for i:=0; i<=m; i++ {
        dp[i] = make([]int, n+1)
    }
    // 填表
    for i:=0; i<m; i++ {
        for j:=0; j<n; j++ {
            if i == 0 && j == 0 {
                // 初始化dp[1][1] = 1
                dp[i+1][j+1] = 1
            }else {
                dp[i+1][j+1] = dp[i][j+1] + dp[i+1][j]
            }
        }
    }
    return dp[m][n]
}
```
