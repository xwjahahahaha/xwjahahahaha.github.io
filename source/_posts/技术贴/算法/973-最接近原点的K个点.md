## 题目描述

#### [973. 最接近原点的 K 个点](https://leetcode-cn.com/problems/k-closest-points-to-origin/)

难度中等141

我们有一个由平面上的点组成的列表 `points`。需要从中找出 `K` 个距离原点 `(0, 0)` 最近的点。

（这里，平面上两点之间的距离是欧几里德距离。）

你可以按任何顺序返回答案。除了点坐标的顺序之外，答案确保是唯一的。

 <!-- more -->

**示例 1：**

```
输入：points = [[1,3],[-2,2]], K = 1
输出：[[-2,2]]
解释： 
(1, 3) 和原点之间的距离为 sqrt(10)，
(-2, 2) 和原点之间的距离为 sqrt(8)，
由于 sqrt(8) < sqrt(10)，(-2, 2) 离原点更近。
我们只需要距离原点最近的 K = 1 个点，所以答案就是 [[-2,2]]。
```

**示例 2：**

```
输入：points = [[3,3],[5,-1],[-2,4]], K = 2
输出：[[3,3],[-2,4]]
（答案 [[-2,4],[3,3]] 也会被接受。）
```

**提示：**

1. `1 <= K <= points.length <= 10000`
2. `-10000 < points[i][0] < 10000`
3. `-10000 < points[i][1] < 10000`

## 解题思路及代码

### 第一种方法：根据距离排序

```java
//时间复杂度O(nlogn) O(logn)
class Solution {
    public int[][] kClosest(int[][] points, int K) {
        Arrays.sort(points, new Comparator<int[]>(){
            public int compare(int[] point1, int[]point2){
                return (point1[0]*point1[0] + point1[1]*point1[1]) - (point2[0]*point2[0] + point2[1]*point2[1]);
            }
        });
        return Arrays.copyOfRange(points, 0, K);
    }
}
```

### 第二种方法：优先队列

普通的队列是一种先进先出的数据结构，元素在队列尾追加，而从队列头删除。在优先队列中，**元素被赋予优先级**。当访问元素时，**具有最高优先级的元素最先删除。**优先队列具有最高级先出 （first in, largest out）的行为特征。通常采用**堆**数据结构来实现。

```java
//优先队列
//空间复杂度O(K),
//时间复杂度O(nlogK) 因为队列要维护一个优先顺序，所以插入和排序都需要O(logk)，最坏情况就是n个元素都要插入
class Solution {
    public int[][] kClosest(int[][] points, int K) {
        //新建优先队列。并定义排序规则：由距离从大到小排列
        PriorityQueue<int[]> pq = new PriorityQueue<>(K, new Comparator<int[]>(){
            public int compare(int[] point1, int[] point2){
                return point2[0] - point1[0];
            }
        });

        //将前k个读入
        for(int i=0; i<K; ++i){
            int dist = points[i][0] * points[i][0] + points[i][1] * points[i][1];
            pq.offer(new int[]{dist, i});
        }

        //读后续的点，有比队列头(距离最大)小的就把队列头弹出
        //这里的点不重复，所以不考虑重复
        for(int i=K; i<points.length; ++i){
            int dist = points[i][0] * points[i][0] + points[i][1] * points[i][1];
            if(dist < pq.peek()[0]){
                pq.poll();
                pq.offer(new int[]{dist, i});
            }
        }

        //选取前K个返回
        int[][] ans = new int[K][2];
        for(int i=0; i<K; ++i){
            ans[i] = points[pq.poll()[1]];
        }
        return ans;
    }
}
```
