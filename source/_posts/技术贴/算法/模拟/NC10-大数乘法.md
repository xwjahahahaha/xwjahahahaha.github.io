## 题目描述

[NC10 大数乘法](https://www.nowcoder.com/practice/c4c488d4d40d4c4e9824c3650f7d5571?tpId=188&&tqId=38632&rp=1&ru=/ta/job-code-high-week&qru=/ta/job-code-high-week/question-ranking)

## 描述

以字符串的形式读入两个数字，编写一个函数计算它们的乘积，以字符串形式返回。 

数据范围： 读入的数字大小满足 `0≤n≤1010000≤*n*≤101000`

要求：空间复杂度$O(n)$，时间复杂度 $O(n^2)$

<!-- more -->

示例1:

输入：

```
"11","99"
```

返回值：

```
"1089"
```

说明：

```
11*99=1089 
```

## 示例2

输入：

```
"114514",""
```

返回值：

```
"114514"
```

## 解题思路及代码

```go
package main

/**
 * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
 * 
 * @param s string字符串 第一个整数
 * @param t string字符串 第二个整数
 * @return string字符串
*/
func solve( s string ,  t string ) string {
    lines, carry := make([][]int, 0), 0
    for p1 := len(s)-1; p1 >= 0; p1 -- {
        line := make([]int, 0)
        carry = 0            // 计算每一行时要注意将carry设置为0
        // 处理错位，加0 (除了第一行)
        for i:=0; i<len(s)-1-p1; i++{
            line = append(line, 0)
        }
        for p2 := len(t)-1; p2 >= 0; p2 -- {
            // 计算乘法, 加上上一次的进位
            mul := int(s[p1]-'0') * int(t[p2]-'0') + carry
            // 个位数 
            num := mul % 10 
            // 计算进位
            carry = mul / 10
            // 写入到结果
            line = append(line, num)
        }
        if carry != 0 {
            line = append(line, carry)
        }
        // 加入该行到所有
        lines = append(lines, line)
    }
    // 将结果集合竖直方向累加
    res := ""
    carry = 0
    // 遍历每一列
    for i:=0; ; i++ {
        // 遍历每一行, flag标识是否都到达列尾
        sum, flag := 0, false
        for j:=0; j<len(lines); j++ {
            // 列的长度不同
            if i < len(lines[j]) {
               flag = true        // 一旦有小于列长度的，说明没有全部到达列尾
               sum += lines[j][i]
            }
        }
        // 统计结果
        sum += carry
        // 退出循环的判断: 全部都到达列尾
        if !flag {
            break
        }
        res = string(byte(sum % 10 + '0')) + res
        // 计算进位
        carry = sum / 10
    }
    if carry != 0 {
        res = string(byte(carry + '0')) + res
    }
    return res
}
```
