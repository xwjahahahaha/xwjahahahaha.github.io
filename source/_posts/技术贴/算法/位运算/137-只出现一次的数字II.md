## 题目描述

[137. 只出现一次的数字 II](https://leetcode-cn.com/problems/single-number-ii/)

难度中等

给你一个整数数组 `nums` ，除某个元素仅出现 **一次** 外，其余每个元素都恰出现 **三次 。**请你找出并返回那个只出现了一次的元素。

 <!-- more -->

**示例 1：**

```
输入：nums = [2,2,3,2]
输出：3
```

**示例 2：**

```
输入：nums = [0,1,0,1,0,1,99]
输出：99
```

 

**提示：**

- `1 <= nums.length <= 3 * 104`
- `-231 <= nums[i] <= 231 - 1`
- `nums` 中，除某个元素仅出现 **一次** 外，其余每个元素都恰出现 **三次**

 

**进阶：**你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

## 解题思路及代码

```go
// 位运算
// int数字是32位，如果数组所有数字的位之和的某现一次的数在该一位不能被3整除则说明只出位为1
// 时间复杂度O(32N)空间复杂度O(1) 
func singleNumber(nums []int) int {
    ans := int32(0)
    for i:=0; i<32; i++ {     // i遍历32位
        // 对于每一位计算总和total
        bitTotal := 0
        for _, num := range nums {  // j遍历每个数
            // 每个数的第i位相加
            bitTotal += (num >> i) & 1 // 左移i位
        }
        // 判断是否能被整除，不能被整除则说明一定有数出现了一次（在此位上）
        if bitTotal % 3 != 0 {
            // 给结果对应位设置为1
            ans = ans | (1 << i) 
        }
    }
    return int(ans)
}
```
