## 题目描述

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

说明：

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

示例 1:

输入: [2,2,1]

输出: 1

示例 2:

输入: [4,1,2,1,2]

输出: 4

## 解题思路及代码

```java
class Solution {
    //第一种方法：异或运算特性：
    // x ^ x = 0
    // 0 ^ x = x
    // 线性复杂度、无额外空间
    public int singleNumber(int[] nums) {
        int res = 0;
        for(int num: nums){
            res ^= num;
        }
        return res;
    }


    //方法二：HashMap存储
    //时间复杂度O(N),空间复杂度O(N)
    public int singleNumber(int[] nums) {
        Map<Integer,Integer> map = new HashMap<>();
        //记录
        for(int num: nums){
            Integer times = map.getOrDefault(num, 0);
            map.put((Integer)num, times+1);
        }
        //查询单个次数
        for(Integer k: map.keySet()){
            if(map.get(k) == 1)
                return k;
        }
        return -1;
    }
}
```
